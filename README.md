# TamagochiPuchhy
Juego de Tamagochi Puchhy

En el juego se puede alimentar a Puchhy, Jugar con él. Hacerlo dormir, despertarlo. Si se pone muy raro se lo puede curar. 
Todas las acciones le cuestan vida, y le restan Energía.
